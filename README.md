# C++ ATM System

A console-based ATM simulation system built using C++ and Object-Oriented Programming (OOP).

## Features

- User login with PIN authentication
- Create a new account
- Check account balance
- Deposit money
- Withdraw money
- Change PIN
- Multiple user accounts
- ATM lock after failed PIN attempts

## Concepts Used

- Classes and Objects
- Encapsulation
- Constructors
- Member Functions
- Arrays of Objects
- Loops and Conditional Statements
- User Input Handling

## How It Works

The program allows users to log in using their name and PIN.  
After successful authentication, users can perform ATM operations such as checking balance, depositing, withdrawing, and changing their PIN.

## Technologies

- C++
- Standard Library (iostream, string)
